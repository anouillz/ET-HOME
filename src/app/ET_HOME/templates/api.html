ET-API

API functions :

  • /transactions
    • Get transactions per date ex : http://localhost:8000/api/transactions/2024-10-22/
    • Get all transactions ex : http://localhost:8000/api/transactions/all/
  • /spendings
    • Get all the spending from one category ex : http://localhost:8000/api/spendings/123e4567-e89b-12d3-a456-426614174000/
